# SmartGrant AI

**Проект для Young Scientist Hackathon 2025** (при V Конгрессе Молодых Ученых)

SmartGrant AI — инновационное решение для управления грантовым финансированием с использованием смарт-контрактов и искусственного интеллекта. Проект направлен на создание прозрачной системы мониторинга и распределения грантов, которая минимизирует бюрократию и повышает эффективность контроля за целевым использованием средств.

## Ключевые особенности

- **Смарт-контракты** для автоматизации и прозрачности грантового финансирования
- **ИИ в финтехе** — основная инновация проекта, позволяющая анализировать заявки, прогнозировать эффективность расходования средств и автоматизировать принятие решений
- **Снижение бюрократической нагрузки** на грантополучателей и грантодателей
- **Прозрачный мониторинг** расходования средств в режиме реального времени

## Требования

- Python 3.13
- Windows
- Зависимости из requirements.txt

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:

```
pip install -r requirements.txt
```

3. Создайте файл конфигурации `config.py` в корневой директории проекта со следующим содержимым:

```
NVIDIA_API_SC   = "ваш_nvidia_api_ключ_sc"
NVIDIA_API_BILL = "ваш_nvidia_api_ключ_bill"
```

## Структура проекта

```
smartgrant-ai/
│
├── config.py          # Файл конфигурации с API-ключами (создается вручную)
├── requirements.txt   # Зависимости проекта
├── main.py           # Главный файл запуска
└── README.md         # Этот файл
```

## Запуск

Запустите проект командой:

```
streamlit run main.py
```
