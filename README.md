# SmartGrant

**Проект для Young Scientist Hackathon 2025** (при V Конгрессе Молодых Ученых)

SmartGrant — инновационное решение для управления грантовым финансированием с использованием смарт-контрактов и искусственного интеллекта. Проект направлен на создание прозрачной системы мониторинга и распределения грантов, которая минимизирует бюрократию и повышает эффективность контроля за целевым использованием средств.

## Ключевые особенности

- **ИИ в финтехе** — основная инновация проекта, позволяет повысить эффективность человеческого труда
- **Снижение бюрократической нагрузки** на грантополучателей и грантодателей
- **Прозрачный мониторинг** использования средств гранта

## Требования

- Python 3.13
- Windows
- Зависимости из requirements.txt
- Доступ к Nvidia API

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:

```cmd
pip install -r requirements.txt
```

3. Создайте файл конфигурации `config.py` в корневой директории проекта со следующим содержимым:

```python
LLM_API_URL = "ВАШ API-КЛЮЧ NVIDIA"
```

## Структура проекта

```
smart-grant/
```
